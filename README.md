# Application Security Write-Ups

This repository contains write-ups for various application security challenges and vulnerabilities. This is an academic repository created for an Application Security course, and it is intended for educational purposes only. The write-ups cover a range of topics, including web application vulnerabilities, secure coding practices, and penetration testing techniques.

> Due to some restrictions, a part of content and discussions shall remain private and will not be shared publicly over here.
